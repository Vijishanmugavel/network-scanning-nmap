# 🔐 Network Scanning and Service Enumeration using Nmap

## 📌 Overview
This project demonstrates basic network scanning and service enumeration using Nmap in a Kali Linux virtual lab environment.

## 🧪 What was done
- Performed a scan on localhost (127.0.0.1)
- Observed all ports in a closed state initially
- Enabled SSH service and identified an open port (22/tcp)
- Conducted service version detection using Nmap

## 🧠 Key Learning
This project helped in understanding how open ports and running services can be identified and analyzed as part of basic security assessment.

## 🛠️ Tools Used
- Kali Linux
- Nmap
